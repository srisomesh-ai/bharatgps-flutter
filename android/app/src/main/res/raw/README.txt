Place your bundled alert sound files here as:
  siren.mp3
  horn.mp3
  chime.mp3
  alarm.mp3
  beep.mp3

Requirements:
- Short (1-5 seconds), small files
- Lowercase names, no spaces, letters/numbers/underscore only
- mp3, ogg, or wav

'default' uses the system default sound (no file needed).
The notification channel for each sound is created automatically by the app.
